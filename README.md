# Genome Assembler
This is a genome assembler (GA) written in python.


## Installation
To take a copy of our GA you should run this command in terminal

`git clone https://github.com/Serfentum/genome_assembler.git`


## Prerequisites
1. python3.x interpretator
2. biopython
3. graphviz
